# 时间管理功能说明

## 功能概述

本次更新为评估系统增加了完整的时间管理功能,包括可视化时间线、时间筛选和历史记录回溯,帮助用户更好地追踪儿童的发展历程。

## 主要功能

### 1. 时间线视图 (TimelineView)

#### 访问方式
- 从档案列表页面,点击档案卡片上的「时间线」按钮
- 从评估报告页面,点击右上角的「历史记录」按钮

#### 视觉特性
- **时间轴设计**: 采用渐变色的时间轴线,从深蓝到浅蓝,清晰展示时间顺序
- **日期分组**: 所有评估记录按日期自动分组,每天显示为一个节点
- **卡片式展示**: 每次评估以卡片形式展示,包含完整信息

### 2. 时间筛选功能

#### 时间范围筛选
提供 5 种时间范围选项:
- **全部**: 显示所有历史记录
- **近一周**: 最近 7 天的评估记录
- **近一月**: 最近 30 天的评估记录
- **近三月**: 最近 90 天的评估记录
- **近一年**: 最近 365 天的评估记录

#### 量表类型筛选
支持按量表类型筛选:
- **全部量表**: 显示所有量表的评估记录
- **简易量表**: 仅显示简易社交观察量表记录
- **SRS**: 仅显示社交反应量表记录
- **SNAP-IV**: 仅显示 SNAP-IV 量表记录
- **ABC**: 仅显示 ABC 量表记录
- **CARS**: 仅显示 CARS 量表记录

### 3. 趋势分析

#### 趋势指示器
对于每种量表,系统会自动计算发展趋势:
- 📈 **改善**: 最新得分低于初次得分(正向变化)
- 📉 **需关注**: 最新得分高于初次得分(需要关注)
- ➡️ **稳定**: 得分保持不变

#### 得分可视化
- 使用进度条直观显示得分占比
- 根据评估结果显示不同颜色的进度条
- 绿色: 正常范围
- 橙色: 轻度异常
- 红色: 严重异常

### 4. 历史记录回溯

#### 快速跳转
- 点击时间线上的任意评估卡片,即可查看该次评估的详细报告
- 从报告页面可以返回时间线继续浏览

#### 详细信息展示
时间线卡片显示以下信息:
- 量表名称和简称
- 评估时间(日期和具体时间)
- 总分和评估等级
- 得分趋势指示器
- 可视化进度条

## 使用场景

### 场景 1: 长期追踪
家长可以查看孩子从第一次评估到最近一次评估的完整历史,了解孩子的整体发展趋势。

### 场景 2: 干预效果评估
通过对比不同时间段的评估结果,家长和专业人士可以评估干预措施的效果。

### 场景 3: 重点关注
使用时间筛选功能,快速查看特定时间段内的评估变化,关注短期内的进展或问题。

### 场景 4: 跨量表对比
切换不同的量表筛选器,对比孩子在社交、注意力、情绪等多个维度的发展情况。

## 技术实现

### 数据结构
```typescript
// 评估结果数据结构
interface AssessmentResult {
  id: string;
  profileId: string;
  scaleId?: string;
  date: number;  // Unix 时间戳
  answers: Record<number, number>;
  score: number;
}
```

### 核心功能
- **智能分组**: 按日期自动分组评估记录
- **动态筛选**: 实时响应筛选条件变化
- **趋势计算**: 自动计算和分析得分趋势
- **性能优化**: 使用 useMemo 优化渲染性能

### UI/UX 设计
- 响应式设计,适配移动端和桌面端
- 流畅的动画效果,提升用户体验
- 清晰的视觉层次,信息一目了然

## 注意事项

1. 所有评估数据存储在本地浏览器(localStorage),请定期备份重要数据
2. 删除档案会同时删除该档案的所有评估记录
3. 时间线视图仅显示有评估记录的日期
4. 趋势分析需要至少 2 次同一量表的评估记录
