## 项目简介

孤独症儿童量表辅助系统是一款基于 Web 的儿童发育行为评估工具,旨在为家长、教育工作者和医疗专业人员提供便捷、专业的评估服务。系统集成了多个标准化评估量表,并通过 AI 技术提供个性化的评估建议和干预指导。

## 主要功能

### 📊 多维度评估量表
系统内置了 5 个专业评估量表,涵盖儿童发育的不同领域:

| 量表名称 | 用途 | 题目数 | 适用场景 |
|---------|------|---------|---------|
| 简易社交观察量表 | 快速筛查基础社交互动能力 | 5 | 初步筛查 |
| 社交反应量表 (SRS) | 全面评估社交障碍严重程度 | 65 | 深度评估 |
| SNAP-IV 量表 | 筛查注意力缺陷及多动/冲动行为 | 26 | ADHD 筛查 |
| ABC 量表 | 评估孤独症行为特征 | 57 | 行为评估 |
| 卡氏量表 (CARS) | 评估孤独症严重程度 | 15 | 临床诊断辅助 |

### ⏱️ 时间管理与历史追踪
- **可视化时间线**: 清晰展示所有评估记录的时间顺序
- **智能筛选**: 按时间范围(近一周/一月/三月/一年)和量表类型筛选
- **趋势分析**: 自动分析得分变化趋势(改善/稳定/需关注)
- **快速回溯**: 点击任意记录查看详细报告

### 🤖 AI 智能助手
- **多平台支持**: 集成硅基流动、算能平台等国内主流大模型
- **个性化建议**: 根据评估结果生成温暖、专业的干预建议
- **流式响应**: 实时显示 AI 生成的建议内容
- **灵活配置**: 支持自定义接入任何 OpenAI 兼容的 API 服务

### 📋 评估报告
- **详细分析**: 显示得分、评估等级、各项得分明细
- **专业解读**: AI 助手提供个性化的评估反馈
- **多格式导出**: 支持导出为图片和 PDF 格式
- **历史对比**: 支持查看历史记录和趋势变化

### 👶 儿童档案管理
- **多档案支持**: 为不同儿童创建独立档案
- **信息记录**: 记录姓名、年龄、基本情况等信息
- **档案管理**: 支持编辑和删除档案

## 使用指南

### 首次使用

1. **创建儿童档案**
   - 点击「添加新档案」
   - 填写儿童姓名、年龄和基本情况
   - 保存档案

2. **配置 AI 助手**(可选)
   - 点击右上角 ⚙️ 设置图标
   - 选择一个平台(如硅基流动)
   - 输入 API Key 并选择模型
   - 保存配置

3. **开始评估**
   - 选择档案 → 点击「开始评估」
   - 选择评估量表
   - 根据题目描述选择合适的选项
   - 提交评估

4. **查看报告**
   - 查看评估得分和等级
   - 阅读 AI 助手的个性化建议
   - 导出报告为图片或 PDF

### 查看历史记录

1. **进入时间线**
   - 在档案列表点击「时间线」按钮
   - 或在报告页面点击「历史记录」

2. **筛选记录**
   - 选择时间范围(近一周/一月等)
   - 选择量表类型
   - 查看符合条件的评估记录

3. **查看详情**
   - 点击任意评估卡片查看详细报告
   - 查看趋势分析

## 数据隐私与安全

### 本地存储
- 所有评估数据存储在浏览器本地(localStorage)
- 不会上传到任何第三方服务器
- 儿童信息完全在本地处理

### API Key 安全
- API Key 仅保存在浏览器本地
- 不会传输到除 AI 平台外的任何服务
- 建议定期更换 API Key
- 请勿在公共设备上使用

### 数据备份
建议定期导出重要评估报告,避免因浏览器数据清理导致数据丢失。

## 常见问题

### Q1: 如何备份我的数据?

**A**: 目前建议定期导出评估报告为 PDF 或图片格式保存。未来版本将支持数据导出功能。

### Q2: 可以在多台设备上使用吗?

**A**: 目前数据存储在本地浏览器中,不同设备间不共享。可以通过导出报告方式在其他设备查看,但需要重新创建档案。

### Q3: AI 建议是否可靠?

**A**: AI 建议基于专业医学知识和标准化量表,仅供参考。不能替代专业医疗诊断。如有疑虑,请咨询专业医生。

### Q4: 为什么需要配置 AI?

**A**: AI 功能需要调用大模型 API 生成个性化建议。不配置 AI 仍可完成评估和查看报告,但不会获得 AI 建议部分。

### Q5: 如何保证数据隐私?

**A**: 所有数据存储在浏览器本地,不会上传到服务器。AI 调用时仅发送匿名化的评估数据,不包含个人身份信息。
