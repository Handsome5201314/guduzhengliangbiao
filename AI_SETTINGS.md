# AI 设置说明

## 功能概述

本系统支持接入多个主流大模型平台的API,为您提供灵活的AI助手服务。您可以根据需求选择不同的平台和模型,确保获得最佳的使用体验。

## 支持的平台

### 1. 硅基流动 (SiliconFlow)

**介绍**: 国内领先的 AI 模型推理服务提供商,提供多种主流大模型 API。

**支持模型**:
- DeepSeek-V3
- DeepSeek-R1
- Qwen2.5-72B-Instruct
- Meta-Llama-3.1-70B-Instruct
- Yi-1.5-34B-Chat

**获取 API Key**:
1. 访问 [硅基流动官网](https://siliconflow.cn/)
2. 注册/登录账号
3. 进入控制台 → API Keys
4. 创建新的 API Key

### 2. 算能平台 (Cambricon)

**介绍**: 专注于国产 AI 芯片和模型推理服务的平台。

**支持模型**:
- ChatGLM3-6B
- GLM-4
- GLM-4V

**获取 API Key**:
1. 访问 [算能平台官网](https://api.cambricon.com/)
2. 注册/登录账号
3. 在开发者中心获取 API Key

### 3. 自定义 OpenAI 兼容平台

**介绍**: 支持任何符合 OpenAI API 标准的平台,包括:
- 百度文心一言
- 阿里通义千问
- 腾讯混元
- 智谱 AI (Zhipu AI)
- 月之暗面 (Moonshot AI)
- 以及其他 OpenAI 兼容的 API 服务

**配置方法**:
1. 选择「添加自定义平台」
2. 填写平台信息:
   - **平台显示名称**: 给平台的别名,如「文心一言」
   - **平台标识**: 英文标识,如 `wenxin`
   - **Base URL**: API 的基础地址,如 `https://aip.baidubce.com/v1`
3. 保存后即可使用

## 使用指南

### 访问设置页面

1. 在主页面(儿童档案列表)右上角点击 ⚙️ 设置图标
2. 进入 AI 设置页面

### 配置平台

1. **选择当前使用的平台**:
   - 在页面顶部选择您想使用的平台
   - 选中的平台会显示蓝色边框和 ✓ 标记

2. **输入 API Key**:
   - 在对应平台的输入框中输入您的 API Key
   - 点击 👁️ 图标可查看/隐藏 API Key
   - API Key 会被加密保存在浏览器本地

3. **选择模型**:
   - 从下拉菜单中选择该平台支持的模型
   - 不同平台提供的模型列表可能不同

4. **保存配置**:
   - 点击底部的「保存配置」按钮
   - 保存成功后会显示提示信息

### 切换平台

- 在任何时间都可以切换到已配置的其他平台
- 切换后新生成的 AI 建议将使用新平台
- 历史数据不受影响

### 添加自定义平台

1. 滚动到页面底部
2. 在「添加自定义平台」区域填写信息:
   - 平台显示名称(中文)
   - 平台标识(英文)
   - Base URL(API 地址)
3. 点击「添加平台」按钮
4. 在新添加的平台中输入 API Key 并选择模型

### 删除平台配置

1. 点击对应平台配置卡片右上角的 🗑️ 图标
2. 确认删除操作
3. 系统默认的平台无法删除

## 常见问题

### Q1: 为什么配置后没有生成 AI 建议?

**A**: 请检查以下几点:
1. 确保已正确输入 API Key
2. 确保已选择一个模型
3. 确保当前选中的平台显示为已配置状态
4. 检查网络连接是否正常
5. 确认 API Key 是否有效且未过期

### Q2: 如何测试 API Key 是否有效?

**A**: 完成一次评估后,系统会自动调用 AI 服务生成建议。如果失败,会显示错误信息提示您检查配置。

### Q3: 不同平台的效果有什么区别?

**A**: 不同模型的特点:
- **DeepSeek**: 推理能力强,适合复杂问题
- **Qwen**: 中文理解优秀,适合中文场景
- **GLM**: 对话流畅,适合互动交流
- **Llama**: 英文能力强,适合双语场景

建议根据实际需求选择最适合的模型。

### Q4: API Key 安全吗?

**A**:
- API Key 仅保存在您浏览器的本地存储(localStorage)中
- 不会上传到任何第三方服务器
- 请勿在公共设备上使用
- 建议定期更换 API Key

### Q5: 如何查看当前使用的平台和模型?

**A**: 在设置页面顶部可以查看:
- 当前选中的平台(有蓝色边框标记)
- 该平台配置的模型
- 其他平台的配置状态

### Q6: 可以同时使用多个平台吗?

**A**:
- 可以配置多个平台的 API Key
- 但同一时间只能激活一个平台
- 可以随时在设置中切换使用

## 技术细节

### 数据存储

- 所有配置数据存储在浏览器本地
- 使用 localStorage 持久化保存
- 清除浏览器数据会丢失配置

### API 调用

- 支持 OpenAI 兼容的 API 格式
- 支持流式响应(streaming)
- 自动处理网络错误和重试

### 费用说明

- AI API 调用费用由平台直接收取
- 具体费率请参考各平台的定价页面
- 建议设置消费预警
